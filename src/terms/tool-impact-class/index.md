<section class="about">
  <h2>Tool Impact Class</h2>
  <p>
    Screwdriver bits and power drivers are designed to handle different levels of torque. Using non-impact bits 
    in an impact driver leads to premature wear or breakage, while torsion-zone bits are specifically engineered 
    to absorb shock loads.
  </p>
  <p>
    This classification provides a shared reference for selecting appropriate tooling based on machine type and load conditions.
  </p>
</section>
<section>
  <h2>Related term set</h2>

  </p>
</section>
